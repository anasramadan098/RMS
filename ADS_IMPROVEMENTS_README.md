# تحسينات نظام إدارة الإعلانات متعدد اللغات (Multilingual Ads Management System)

## المزايا الجديدة المضافة

### 1. حقول قاعدة البيانات الجديدة (متعددة اللغات)
- **اسم الإعلان العربي (name_ar)**: حقل نصي لعنوان الإعلان باللغة العربية
- **اسم الإعلان الإنجليزي (name_en)**: حقل نصي لعنوان الإعلان باللغة الإنجليزية
- **وصف الإعلان العربي (description_ar)**: وصف مختصر باللغة العربية
- **وصف الإعلان الإنجليزي (description_en)**: وصف مختصر باللغة الإنجليزية
- **مسار الصورة الإنجليزية (path_en)**: صورة منفصلة للغة الإنجليزية (اختياري)
- **وقت البداية (start_time)**: توقيت بداية عرض الإعلان
- **وقت النهاية (end_time)**: توقيت انتهاء عرض الإعلان
- **حالة التفعيل (active)**: حقل boolean للتحكم في تفعيل/إلغاء الإعلان

### 2. تحسينات نموذج Ad (متعدد اللغات)
- إضافة جميع الحقول الجديدة إلى `$fillable`
- إضافة `casts` لتحويل الأوقات والقيم المنطقية تلقائياً
- دالة `isActiveNow()` محسّنة للتحقق من حالة النشاط والتوقيت
- دالة `getName($locale)` للحصول على الاسم حسب اللغة
- دالة `getDescription($locale)` للحصول على الوصف حسب اللغة
- دالة `getImagePath($locale)` للحصول على مسار الصورة حسب اللغة

### 3. تطوير المتحكم (AdsController) متعدد اللغات
- **إنشاء إعلان جديد**: دعم رفع صورتين منفصلتين (عربي/إنجليزي)
- **تعديل الإعلان**: إمكانية تحديث كل لغة منفصلة
- **حذف الإعلان**: حذف جميع الصور المرتبطة
- **عرض الإعلانات**: فلترة الإعلانات النشطة فقط
- **التحقق من صحة البيانات**: قواعد محسّنة للحقول متعددة اللغات

### 4. واجهات المستخدم المحسّنة (متعددة اللغات)

#### صفحة القائمة (index.blade.php)
- عرض الأسماء والأوصاف حسب اللغة الحالية
- مؤشر حالة تفعيل الإعلان
- عرض حالة النشاط الزمني
- تصميم بطاقات محسّن

#### صفحة الإنشاء (create.blade.php)
- نماذج منفصلة للمحتوى العربي والإنجليزي
- رفع صور منفصلة لكل لغة
- معاينة فورية للصور
- مفتاح تفعيل/إلغاء الإعلان
- التحقق من صحة الأوقات

#### صفحة التعديل (edit.blade.php)
- عرض الصور الحالية لكل لغة
- إمكانية تحديث المحتوى لكل لغة منفصلة
- تغيير الصور اختياري

#### صفحة العرض (show.blade.php)
- تبويبات لعرض الصور بكل لغة
- عرض المحتوى بكل اللغات
- أكورديون لعرض الأوصاف بكل اللغة
- أزرار تحميل منفصلة لكل صورة
- عرض حالة التفعيل

### 5. تحسينات إضافية

#### إدارة الصور متعددة اللغات
- نظام تسمية ملفات محسّن (مع بادئة ar/en)
- ضغط الصور بجودة 85%
- حذف آمن للصور عند التحديث/الحذف

#### نظام اللغات
- التبديل التلقائي حسب لغة التطبيق
- عرض احتياطي للغة الأخرى عند عدم وجود محتوى
- دعم RTL و LTR

## التحقق من صحة البيانات المحسّن

### قواعد التحقق الجديدة:
- **الصورة العربية**: إجبارية، أنواع مدعومة، حد أقصى 2MB
- **الصورة الإنجليزية**: اختيارية، نفس القيود
- **الاسم العربي**: إجباري، نص، حد أقصى 255 حرف
- **الاسم الإنجليزي**: إجباري، نص، حد أقصى 255 حرف
- **الأوصاف**: اختيارية، نص، حد أقصى 1000 حرف
- **الأوقات**: إجبارية، تنسيق HH:MM، وقت النهاية بعد البداية
- **حالة التفعيل**: قيمة منطقية

## استخدام النظام متعدد اللغات

### 1. إنشاء إعلان جديد:
1. اذهب إلى `/ad/create`
2. املأ الاسم والوصف لكل لغة
3. ارفع الصورة العربية (إجباري)
4. ارفع الصورة الإنجليزية (اختياري)
5. حدد أوقات العرض
6. اختر حالة التفعيل
7. اضغط "إنشاء الإعلان"

### 2. تعديل إعلان:
1. من قائمة الإعلانات، اضغط "تعديل"
2. عدّل المحتوى لكل لغة
3. غيّر الصور إذا لزم الأمر
4. اضغط "تحديث الإعلان"

### 3. عرض الإعلان:
- استخدم التبويبات لعرض صور كل لغة
- اعرض الأكورديون لرؤية المحتوى بكل اللغات
- حمّل الصور منفصلة

## المزايا التقنية

- **دعم متعدد اللغات**: عربي وإنجليزي كامل
- **إدارة صور متقدمة**: صور منفصلة لكل لغة
- **أداء محسّن**: استعلامات محسّنة وcaching
- **أمان**: التحقق الشامل من جميع المدخلات
- **سهولة الاستخدام**: واجهات بديهية متعددة اللغات
- **مرونة**: عرض احتياطي عند عدم وجود محتوى
